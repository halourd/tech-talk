### Run Locally

Clone the project

```bash
  git clone https://github.com/halourd/tech-talk.git
```

Go to the project directory

```bash
  cd tech-talk
```

Install dependencies

```bash
  npm install
```
Open project folder

```bash
  code .
```

Start the server using terminal

```bash
  npx expo start
```
